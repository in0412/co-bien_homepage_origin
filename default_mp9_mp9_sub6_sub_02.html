인터페이스 > EAI/B2Bi

• WebMethods - EAI/B2Bi


webMethods 아키텍처는 Integration Server (엔진), Universal Messaging(메시징 허브), Adapter를 포함한 여러 기능 요소로 구성되어 있으며, 특히 단일의 엔진인 Integration Server가 EAI와 B2B 모두를 지원하여 확장성과 관리성이 우수합니다.
Cloud(SaaS, PaaS), IoT, Big Data와의 연계, 그리고 오픈 API로의 확장을 위한 포괄적인 통합기능(Hybrid Integration)을 제공하고 있으며, 해당 영역에서 최고의 솔루션으로 평가되고 있습니다.



기능 요약


- 30개 이상의 어댑터를 이용한 어플리케이션 연계 (SAP, Siebel, DB 등)
- 국제표준의 프로토콜 및 문서형식을 지원 (http(s), FTP, SFTP, SOAP ,XML, SON, EDI , 로제타넷, ebXML 등)
- 코드 작성 없이 개발툴을 통해 손쉽고 빠르게 서비스 생성
- 기 생성된 서비스를 Drag & Drop하여 재활용 가능



활용 방안


- SAP와 내부 Legacy System 연계
- Legacy System 간의 연계
- 거래처, 고객 또는 VAN사와 EDI, 로제타넷, ebXML 연계
- 모바일 환경 연계 (IOS,Android, windows)
- 웹과의 유연한 연계 (HTML5 Web Socket, Silverlight, 자바 등)
